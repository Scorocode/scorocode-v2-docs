    Как установить sc-cli

    Scorocode CLI (command line interface, также известен как консоль) - разновидность текстового интерфейса между человеком и компьютером.

    Для того чтобы установить sc-cli, скачайте инсталлятор

    Скачать инсталлятор sc-cli для Mac OS

    Скачать инсталлятор sc-cli для Debian

    Скачать инсталлятор sc-cli для Windows

    После установки sc-cli введите в командной строке sc-cli. Из адресной строки приложения скопируйте идентификатор приложения (32 символа), с помощью команды fetch инициализируйте работу cli для выбранного приложения, через bridge свяжите порты.

    pull и push скачивают файлы приложения на локальный компьюьтер и загружают их в облако.

    serve симулирует node.js на локальном ПК, dep обновляет зависимости node.js

    logs выводит логи, regdb регистрирует БД, которую можно редактировать
